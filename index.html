Rally ALM Tasks Integration

Version 0.1
rally-intellij-plugin.zip

Instructions

  • Install at Preferences -> Plugins -> "Install Plugin from Disk"
  • Restart IntelliJ
  • Configure Preferences -> Rally

Misconfiguring the Rally section or failing to configure it before use might result in undesirable behavior, as not all exception conditions are well handled.

Known limitations

  • I block the ui-thread on the first time clicking into the 'Rally Search' window on the bottom to cache the initial set of projects. This takes almost a minute to load.
  • If 'Rally Search' throws an exception any reason first opening the window, the IDE needs restarting for it to try again. Possible ways to cause an exception include an auth failure or a misconfigured Rally url.
  • The html rendering on an artifact's description and notes isn't perfect; it gets confused by complicated markup.
  • Images embedded in a description don't display
